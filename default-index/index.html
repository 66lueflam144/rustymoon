SSRF–服务端请求伪造

服务端伪造请求（Server-Side Request Forgery，SSRF）是一种安全漏洞，攻击者通过操纵服务器发起的请求，使其访问目标服务器或资源，从而在服务端执行未经授权的操作。这种攻击通常发生在应用程序未正确验证或过滤用户提供的输入数据时。
